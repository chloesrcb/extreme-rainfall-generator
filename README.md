This repository contains the code associated with the paper 
"Spatio-temporal modeling of urban extreme rainfall events at high resolution" from SERRE-COMBE, MEYER, OPITZ and TOULEMONDE.

It implements a novel spatio-temporal stochastic model for urban rainfall extremes,
combining EGPD marginals with a flexible extremal dependence structure based on r-Pareto processes with advection incorporation.
The model is calibrated on the Montpellier OMSEV precipitation network at high spatio-temporal resolution.
